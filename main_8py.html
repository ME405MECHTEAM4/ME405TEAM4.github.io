Detailed Description

This is the main file that sends data to stepper.py and task_motor.py to control the dc and the two stepper motors.

Here we parse through an HPGL file that we created by drawing a shape in inkscape. The parse function creates three arrays for x and y location as well as pen up/down commands. From there, these arrays are put into a queue that is read in the motor_task function which further interpolates our data to create bytearrays that are to be pushed to our stepper.py file
